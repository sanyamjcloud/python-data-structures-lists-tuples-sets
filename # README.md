# Python Calculator Using Functions

This project demonstrates the use of Python functions to create a simple calculator.

## Features
- Add, subtract, multiply, divide operations
- Functions with parameters and return values
- Default arguments
- Division by zero handling
- Menu-based user input
- Proper docstrings
- Independent testing of functions

## Files
- calculator.py – Calculator implementation
- README.md – Project explanation

## How to Run
Open terminal and execute:
```bash
python calculator.py
